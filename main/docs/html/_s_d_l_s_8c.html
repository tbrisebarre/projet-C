void SDLS_displayAll ( )

use this function after creating shape and texture to display in window
use of double buffering : all operation before this function are put in video RAM but not displayed

1 SDLS_init("Hello World");
2  SDLS_setColor(255,0,0); // red color
3  SDLS_drawLineRect(50,50,100,20); // rectangle in 50,50 size 100x20
4  SDLS_displayAll(); // copy cached Video Buffer to Video buffer
5  SDL_Delay(3000); // wait 3s to see the window
6  SDLS_cleanup(); // erase memory

getting an array of pixel of window displayed
You can use this function with SDLS_putpixels and change pixels of your window

1 // darking a zone on position 100,100 and size 20,100 of window. You need to init the window before
2 Uint32 * pixels;
3 pixels = (Uint32*) malloc(SDLS_getScreenHeight()*SDLS_getScreenHeight()*sizeof(Uint32)); // creation d'un tableau de pixel à la taille de la fenêtre
4 SDLS_getpixels(pixels,SCREEN_WIDTH,SCREEN_HEIGHT); // copy de la fenêtre dans le tableau pixels
5 for(int i=0;i<20;i++)
6  for(int j=0;j<100;j++)
7  {
8  int val = pixel[100+i][100+j]; // recup pixel position 100+i et 100+j (rectangle de 20 par 100)
9  int cr = ((val)&0xFF0000) >> 16; // récup couleur rouge du pixel
10  int cg = ((val)&0xFF00) >> 8; // recup couleur verte du pixel
11  int cb = ((val)&0xFF); // recup couleur bleue du pixel
12  cr >>= 1 ; // division par 2 de la valeur de la couleur, on assombri cette zone
13  cg >>= 1 ;
14  cb >>= 1 ;
15  }
16  SDLS_putpixels(pixels,SCREEN_WIDTH,SCREEN_HEIGHT);
17 free(pixels);
Parameters
pixelsarray of pixel from window
wwidth of pixel to get
hheight of pixel to get
Returns
number of pixels copied
